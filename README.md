catvideo by Erica and Yun 
==========
INSTALL INSTRUCTION

When you are downloading the file, make sure you put the file in htdocs so you can view the website with the address:
http://localhost/.../index.php

* ... represents the file name which you put the files in.

To source the database:
source .../catVideosDB.sql

* ... represents the file path of your file catVideoDB.sql
example: source C:\xampp\htdocs\catvideo\catVideoDB.sql
